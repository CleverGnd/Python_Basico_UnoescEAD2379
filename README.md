# Curso de Introdução à Programação
Repositório criado para apresentar e disponibilizar as atividades desenvolvidas durante o curso de Introdução à Programação.

Neste curso, foi possível aprender os conceitos básicos de programação, como variáveis, estruturas de controle, laços de repetição e funções, utilizando a linguagem Python. Além disso, foram apresentados alguns conceitos de lógica de programação, como algoritmos e estruturas de dados.

Este repositório está organizado em pastas, cada uma delas contendo as unidades desenvolvidas durante as aulas. Além disso, é possível encontrar no diretório principal algumas referências e materiais de apoio utilizados durante o curso.

Espero que este repositório possa ser útil para outros estudantes de programação e que possa servir como uma base para o desenvolvimento de novos projetos. Caso tenha alguma dúvida ou sugestão, não hesite em entrar em contato.

    - Subunidade 3 - Explorando Tipos de Dados no Python
